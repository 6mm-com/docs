<h2 id="minimal-example">Minimal example</h2>

```html
<div id="trading-widget" style="width: 100%; height: 720px"></div>

<script src="https://app.6mm.com/sdk/trading-widget/v1/index.js"></script>
<script>
  const widget = TradingWidget.create('#trading-widget', {
    baseUrl: 'https://app.6mm.com',
    symbol: 'BTCUSDT',
    theme: 'dark',
    locale: 'en',
    layout: 'default',
    display: {
      showHeader: false,
      showTicker: false,
      hidden: [],
    },
    onReady(event) {
      console.log('ready', event)
    },
    onError(error) {
      console.error('error', error)
    },
  })
</script>
```

<h2 id="sdk-urls">SDK URLs</h2>

| Environment | Script URL |
| --- | --- |
| Production | https://app.6mm.com/sdk/trading-widget/v1/index.js |

> **Set baseUrl explicitly**
>
> When testing from partner-owned pages, pass baseUrl so the iframe always points to the 6MM Web domain.

<h2 id="display-controls">Display controls</h2>

Partner embeds hide the PC header and bottom market ticker by default. Use `display` when you need to show those shell areas or hide high-level containers in the embedded trading page.

```js
display: {
  showHeader: false,
  showTicker: false,
  hidden: ['account'],
}
```
