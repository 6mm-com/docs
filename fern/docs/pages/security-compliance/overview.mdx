<h2 id="recommended-reading-order">Recommended reading order</h2>

1. Start with Security Architecture and Partner Responsibility Boundary.
2. Review API Key Management, Request Signing, and Webhook idempotency before backend integration.
3. Review Compliance Overview and License Support before regional launch preparation.
4. Review legal pages and the launch checklist before exposing trading functionality to users.
