6MM provides technology infrastructure and compliance-oriented support for authorized partners. Compliance requirements vary by jurisdiction and business model, so each launch should be reviewed against the partner's own obligations.

<h2 id="what-this-section-covers">What this section covers</h2>

| Area | Description |
| --- | --- |
| License support | Guidance around supported license paths and required preparation materials. |
| Partner obligations | User onboarding, disclosures, local compliance controls, and customer relationship management. |
| Technical controls | Signing, logging, access control, webhook verification, and production monitoring. |
| Launch review | Checklist-based confirmation before exposing trading functionality to users. |

<h2 id="important-boundary">Important boundary</h2>

6MM documentation helps partners prepare technical and operational controls. It does not replace legal advice, jurisdiction-specific compliance review, or the partner's own regulatory obligations.
