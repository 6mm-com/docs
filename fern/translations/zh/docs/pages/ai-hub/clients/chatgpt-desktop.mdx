ChatGPT Desktop 是普通用户最容易理解的 AI 入口，适合用自然语言询问市场、账户和交易规则。

当用户希望通过对话方式查看公开行情、账户摘要、持仓解释和交易规则上下文时，可以从这个入口开始。

<h2 id="recommended-setup">推荐接入方式</h2>

| 需求 | 方式 |
| --- | --- |
| 公开市场数据 | Hosted MCP |
| 私有账户上下文 | Local MCP |
| 固定分析模板 | Agentic Skills |
| 终端工作流 | 6MM CLI |

<h2 id="what-users-can-ask">用户可以这样问</h2>

```text
使用 6MM 总结 BTC-USDT-PERP 的流动性和资金费率情况。
```

```text
使用 6MM 解释我当前的持仓。不要下单，也不要撤单。
```

```text
使用 6MM 测试网 预览一笔小额 ETH 多单的保证金影响。不要提交订单。
```

<h2 id="safety-defaults">安全默认值</h2>

- 先从公开市场数据开始。
- 需要账户凭证时使用 Local MCP。
- 私有 API key 保留在用户本机。
- 后续任何交易动作都必须先预览并由用户明确确认。
