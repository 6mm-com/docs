MCP 工具 是 AI 客户端可以主动调用的动作。每个 tool 都应包含清晰描述、JSON schema、权限要求、稳定返回格式和错误模型。

<h2 id="phase-1-public-market-tools">Phase 1：公开市场工具</h2>

| Tool | 用途 |
| --- | --- |
| 6mm_system_status | 检查服务和市场可用性。 |
| 6mm_markets | 列出可交易市场和基础元数据。 |
| 6mm_market_summary | 汇总一个或多个交易对的市场状态。 |
| 6mm_ticker | 返回最新 ticker。 |
| 6mm_orderbook | 返回盘口深度。 |
| 6mm_klines | 返回历史 K 线。 |
| 6mm_trades | 返回最近公开成交。 |
| 6mm_funding_rates | 返回当前和历史资金费率。 |

<h2 id="phase-2-account-read-tools">Phase 2：账户只读工具</h2>

| Tool | 用途 |
| --- | --- |
| 6mm_account_summary | 汇总账户状态。 |
| 6mm_balances | 读取余额。 |
| 6mm_positions | 读取仓位。 |
| 6mm_open_orders | 读取当前委托。 |
| 6mm_order_history | 读取历史订单。 |
| 6mm_fills | 读取成交记录。 |
| 6mm_funding_payments | 读取资金费记录。 |

<h2 id="phase-3-and-later-trading-tools">Phase 3 及之后：交易工具</h2>

| Tool | 用途 |
| --- | --- |
| 6mm_preview_order | 校验订单但不下单。 |
| 6mm_create_order | 在确认和风控检查后创建订单。 |
| 6mm_cancel_order | 撤销单个订单。 |
| 6mm_cancel_all_orders | 撤销所有符合条件的当前委托。 |
| 6mm_amend_order | 修改已有订单。 |
| 6mm_order_status | 查询指定订单状态。 |

<h2 id="tool-requirements">Tool 要求</h2>

每个 tool 都应定义：

- 给 AI model 看的描述。
- 输入 JSON schema。
- 权限范围。
- 支持环境。
- 限流行为。
- 稳定返回字段。
- 错误码。
- 审计日志字段。
