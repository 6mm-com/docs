MCP resources 是 AI 客户端在调用工具前可以读取的只读上下文。

Resources 能减少 AI 胡乱猜测，因为模型可以读取稳定、结构化的 6MM 信息。

<h2 id="recommended-resource-uris">推荐 resource URI</h2>

| Resource | 用途 |
| --- | --- |
| 6mm://system/config | 环境和能力配置。 |
| 6mm://system/status | 当前系统和市场状态。 |
| 6mm://markets | 可交易市场。 |
| 6mm://market/{symbol} | 合约规格和市场级元数据。 |
| 6mm://fees | 费用规则。 |
| 6mm://risk/rules | 交易和风险规则摘要。 |
| 6mm://risk/margin | 保证金和清算上下文。 |
| 6mm://funding/rules | 资金费率规则。 |
| 6mm://api/docs | API 参考摘要或链接。 |

<h2 id="resource-design-rules">Resource 设计规则</h2>

- 保持只读。
- 使用稳定 schema。
- 包含时间戳和环境名称。
- 不暴露密钥。
- 优先使用简洁结构化数据，而不是长篇文本。
- 细节过长时链接到权威文档。
