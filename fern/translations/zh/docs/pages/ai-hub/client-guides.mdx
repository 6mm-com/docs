客户端接入指南用于帮助用户把 6MM AI Hub 接入他们已经使用的 AI 工具。

<h2 id="start-with-the-ai-tool-users-already-know">从用户已经认识的 AI 工具开始</h2>

<CardGroup cols={2}>
<Card title="ChatGPT Desktop" icon="fa-duotone fa-message-bot" href="/zh/ai-hub/clients/chatgpt-desktop">
用对话方式查询公开行情、账户摘要和交易规则上下文。
</Card>
<Card title="Claude Desktop" icon="fa-duotone fa-comments" href="/zh/ai-hub/clients/claude-desktop">
通过 Local MCP 或 Hosted MCP 进行研究和账户分析。
</Card>
<Card title="Claude Code CLI" icon="fa-duotone fa-terminal" href="/zh/ai-hub/clients/claude-code-cli">
当 agent 可以执行命令并读取 JSON 输出时，使用 CLI 或 Local MCP。
</Card>
<Card title="Codex CLI" icon="fa-duotone fa-code" href="/zh/ai-hub/clients/codex-cli">
用于 SDK、API、Webhook 和上线任务等实现型工作流。
</Card>
<Card title="Cursor" icon="fa-duotone fa-arrow-pointer" href="/zh/ai-hub/clients/cursor">
把 6MM SDK 和 API 上下文带入编码工作流。
</Card>
<Card title="Gemini CLI" icon="fa-duotone fa-sparkles" href="/zh/ai-hub/clients/gemini-cli">
使用结构化 CLI 输出完成终端型行情研究和自动化。
</Card>
<Card title="VS Code / GitHub Copilot" icon="fa-duotone fa-laptop-code" href="/zh/ai-hub/clients/vscode-copilot">
在编辑器和终端工作流中完成合作方后端接入。
</Card>
<Card title="Windsurf" icon="fa-duotone fa-wind" href="/zh/ai-hub/clients/windsurf">
在能读取项目文件的编码 Agent 工作流中使用 Local MCP 或 CLI。
</Card>
</CardGroup>

<h2 id="each-client-guide-should-include">每篇客户端指南应包含</h2>

- 支持模式：CLI、Local MCP、Hosted MCP 或 Remote MCP。
- 安装命令。
- 配置文件位置。
- 环境变量。
- Testnet 示例。
- 验证命令或提示词。
- 常见错误。
- 安全说明。

<h2 id="example-verification-prompts">验证提示词 示例</h2>

```text
使用 6MM 列出可交易市场。
```

```text
使用 6MM 查询 BTC-USDT-PERP 盘口深度，并总结流动性。
```

```text
使用 6MM 测试网 预览一笔小额 BTC 多单，并解释保证金影响。不要下单。
```
