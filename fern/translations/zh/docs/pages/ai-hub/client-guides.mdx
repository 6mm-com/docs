客户端接入指南用于帮助用户把 6MM AI Hub 接入他们已经使用的 AI 工具。

<h2 id="supported-client-categories">支持的客户端类型</h2>

| 客户端 | 推荐方式 |
| --- | --- |
| Claude Desktop | Local MCP 或 Hosted MCP |
| Claude Code CLI | CLI 或 Local MCP |
| Cursor | Local MCP |
| Codex CLI | CLI 或 Local MCP |
| ChatGPT Desktop | Local MCP 或 Hosted MCP |
| VS Code / GitHub Copilot | Local MCP 或 CLI |
| Gemini CLI | CLI |
| Windsurf | Local MCP |

<h2 id="each-client-guide-should-include">每篇客户端指南应包含</h2>

- 支持模式：CLI、Local MCP、Hosted MCP 或 Remote MCP。
- 安装命令。
- 配置文件位置。
- 环境变量。
- Testnet 示例。
- 验证命令或 prompt。
- 常见错误。
- 安全说明。

<h2 id="example-verification-prompts">验证 prompt 示例</h2>

```text
使用 6MM 列出可交易市场。
```

```text
使用 6MM 查询 BTC-USDT-PERP 盘口深度，并总结流动性。
```

```text
使用 6MM testnet 预览一笔小额 BTC 多单，并解释保证金影响。不要下单。
```
