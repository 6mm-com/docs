AI Hub 应分阶段上线，让用户尽早获得价值，同时保护敏感交易能力。

<h2 id="phase-1-public-market-ai-access">Phase 1：公开市场 AI Access</h2>

目标：让任何支持的 AI 客户端读取 6MM 官方市场数据。

交付：

- 共享 SDK / API client。
- CLI 公开市场命令。
- Local MCP 公开市场工具。
- Hosted MCP 公开市场工具。
- markets、fees、risk rules、system status resources。
- 客户端安装指南。

不包含：

- 账户查询。
- 真实交易。
- API key 托管。

<h2 id="phase-2-account-read">Phase 2：账户只读</h2>

目标：让用户安全读取自己的账户上下文。

交付：

- 只读 API key 权限范围。
- 账户摘要。
- 余额。
- 仓位。
- 当前委托。
- 成交。
- 资金费记录。
- 组合风险提示词。

推荐部署：

- Local MCP。
- CLI。
- Remote MCP self-hosted。

<h2 id="phase-3-trading-preview">Phase 3：交易预览</h2>

目标：让 AI 辅助生成订单，但不执行。

交付：

- preview_order。
- 风险检查。
- 保证金影响。
- 费用估算。
- 清算影响。
- 执行规划提示词。

<h2 id="phase-4-controlled-trading">Phase 4：受控交易</h2>

目标：只在强风控下支持真实交易。

交付：

- create_order。
- cancel_order。
- amend_order。
- cancel_all_orders。
- order_status。
- 幂等键。
- 最大订单金额限制。
- 明确确认。
- 审计日志。

<h2 id="phase-5-agentic-skills">Phase 5：Agentic Skills</h2>

目标：产品化常见市场分析和风险工作流。

交付：

- Market analyst skill。
- Portfolio risk skill。
- Funding research skill。
- Liquidation protection skill。
- Execution planner skill。
- 示例任务库。
