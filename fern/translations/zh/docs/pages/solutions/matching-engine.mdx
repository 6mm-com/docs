白标交易所不只是一个前端外壳。交易、账户、风控和结算背后的核心模块，决定了最终用户体验是否可靠。

<h2 id="core-modules">核心模块</h2>

| 模块 | 作用 |
| --- | --- |
| 撮合与订单生命周期 | 处理下单、撮合状态、撤单和成交记录。 |
| 账户与仓位系统 | 维护余额、保证金状态、仓位和可交易状态。 |
| 风控能力 | 协调杠杆、保证金、清算和异常行情行为。 |
| 行情数据 | 向交易界面分发行情价格、深度、成交和 K 线数据。 |
| 结算与记录 | 对齐交易历史、余额变化和运营记录。 |
