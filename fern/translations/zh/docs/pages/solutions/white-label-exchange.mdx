白标交易所系统面向希望运营自有品牌数字资产交易平台，但不希望独立建设所有交易所核心组件的团队。

<h2 id="included-capability-areas">覆盖能力范围</h2>

- 多终端交易所体验
- 撮合、账户、订单、仓位和风控基础设施
- 流动性连接和市场深度支持
- 品牌和视觉定制
- 运营后台和业务流程支持
- 安全架构与合规规划支持

<h2 id="implementation-focus">实施关注点</h2>

| 领域 | 文档需求 |
| --- | --- |
| 产品上线 | 了解平台范围、用户路径和运营模型。 |
| 品牌交付 | 准备 Logo、入口 Banner 和白标视觉素材。 |
| 交易运营 | 了解订单、保证金、费用和清算机制。 |
| 技术接入 | 连接 API、SDK、Webhook 和监控流程。 |
