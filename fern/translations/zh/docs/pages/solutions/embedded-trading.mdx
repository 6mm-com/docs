嵌入式 API / SDK 交易系统面向已经拥有流量、用户或业务场景的合作方，让合作方无需从零搭建完整交易所，也能快速增加交易能力。

<h2 id="what-it-provides">提供什么能力</h2>

- 通过 Trading Widget SDK 嵌入交易终端
- 通过 Agent SDK 完成 Partner 后端接入
- 用户身份绑定和入口 token 流程
- 资产划转、订单、仓位和账户 API 能力
- 事件、Webhook 和生命周期集成，便于接入运营系统

<h2 id="recommended-architecture">推荐架构</h2>

| 层级 | 职责 |
| --- | --- |
| Partner 前端 | 承载入口并嵌入 6MM 交易界面。 |
| Partner 后端 | 保存 API 密钥、校验用户、签发 embedToken、处理 Webhook。 |
| 6MM 交易基础设施 | 处理交易 UI、撮合、风控、流动性、账户状态和交易生命周期。 |
