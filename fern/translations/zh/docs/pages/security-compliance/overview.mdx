| 领域 | 目的 |

<h2 id="recommended-reading-order">推荐阅读顺序</h2>

1. 先阅读安全架构和合作方责任边界。
2. 后端接入前阅读 API Key 管理、请求签名、Webhook 与幂等。
3. 准备区域上线前阅读合规接入说明和牌照支持。
4. 面向用户开放交易功能前阅读法律文件和上线检查表。
