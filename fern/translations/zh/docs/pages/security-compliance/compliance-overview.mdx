6MM 为授权合作方提供交易技术基础设施和合规导向的支持。合规要求会因司法辖区和业务模式不同而变化，因此每次上线都需要结合合作方自身义务进行确认。

<h2 id="what-this-section-covers">本栏目覆盖内容</h2>

| 领域 | 说明 |
| --- | --- |
| 牌照支持 | 支持路径和准备资料说明。 |
| 合作方义务 | 用户准入、信息披露、本地合规控制和客户关系管理。 |
| 技术控制 | 签名、日志、访问控制、Webhook 验证和生产监控。 |
| 上线审查 | 面向真实用户开放交易功能前的检查项。 |

<h2 id="important-boundary">重要边界</h2>

6MM 文档用于帮助合作方准备技术和运营控制，不替代法律意见、特定司法辖区合规审查，也不替代合作方自身监管义务。
