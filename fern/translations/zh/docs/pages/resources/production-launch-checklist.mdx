在向真实用户开放 6MM 交易能力之前，请使用本检查表确认上线准备情况。

<h2 id="integration-readiness">接入准备</h2>

- 已确认生产域名、跳转路径和回调路径。
- Trading Widget、Agent SDK、API 或 WebSocket 流程已完成端到端测试。
- Partner 后端只在服务端保存所有密钥。
- Webhook 接收、验签、重试和幂等处理已启用。
- 日志中包含 requestId、用户映射 ID、订单 ID 和时间戳。

<h2 id="product-and-resource-readiness">产品与资源准备</h2>

- 交易入口、Banner、卡片素材使用对应语言版本。
- 仅在合作协议允许时使用白标素材。
- 用户侧风险提示和交易入口文案已由合作方确认。
- 已明确支持负责人和升级路径。

<h2 id="operational-readiness">运营准备</h2>

- 监控覆盖 API 错误、Token 发放、Widget 初始化、Webhook 失败和订单生命周期事件。
- 已确认紧急联系路径。
- 已准备交易入口临时关闭或回滚方案。
