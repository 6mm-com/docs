---

新合作方接入或已有接入切换生产前，可以使用本模板整理必要信息。

<h2 id="partner-information">合作方信息</h2>

| 字段 | 说明 |
| --- | --- |
| 合作方名称 | 用于接入配置的公司或项目名称。 |
| 产品类型 | App、钱包、经纪商门户、白标交易所、机构平台或其他。 |
| 主要联系人 | 商务、技术、运营和紧急联系人。 |
| 上线地区 | 接入功能面向的国家或地区。 |

<h2 id="technical-information">技术信息</h2>

| 字段 | 说明 |
| --- | --- |
| 前端域名 | 加载 Widget 或入口页面的域名。 |
| 后端域名 | 调用 Agent SDK 或 API 的服务端域名。 |
| Webhook URL | 生产 Webhook 地址以及是否需要 IP 白名单。 |
| 接入范围 | Trading Widget、Agent SDK、REST API、WebSocket、白标系统或流动性接入。 |

<h2 id="operational-information">运营信息</h2>

- 预计上线时间。
- 预计流量或活跃用户范围。
- 支持时间和升级负责人。
- 监控负责人和事件联系人。
