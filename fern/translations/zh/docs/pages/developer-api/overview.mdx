<h2 id="production-checklist">生产检查</h2>

- API Secret 只保存在后端。
- 签名请求前同步服务端时间。
- 金额、价格、数量等精度敏感字段默认按字符串处理。
- 订单流程需要 requestId、结构化日志和幂等处理。
- 需要实时状态的运营系统应订阅 WebSocket 推送。
- 上线前确认错误码、重试和降级处理策略。
