<h2 id="options">初始化参数</h2>

| 字段 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| baseUrl | string | SDK script 所在域名 | iframe 资源域名，Partner 页面建议显式传入。 |
| symbol | string | BTCUSDT | 会转成大写并移除 - / _。 |
| theme | dark \| light | dark | 初始主题参数。 |
| locale | en \| en-US \| zh-CN | 浏览器或默认语言 | 支持英文和简体中文。 |
| layout | default \| compact | default | 预留布局参数。 |
| auth | object | 无 | 新 SSO 接入建议使用 partner-token。 |

<h2 id="events">事件</h2>

| 事件 | 触发时机 |
| --- | --- |
| ready | iframe 初始化完成。 |
| error | SDK、iframe 或鉴权错误。 |
| state_changed | SDK 生命周期状态变化。 |
| auth_state_changed | partner-token 鉴权状态变化。 |
| symbol_changed | iframe 内实际交易对变化。 |
| locale_changed | iframe 内实际语言变化。 |
| order_created | 用户在 iframe 内成功创建订单。 |
| order_failed | 用户下单失败。 |
| destroyed | SDK 实例销毁。 |
