<h2 id="responsibility-split">职责边界</h2>

| 范围 | 负责人 | 说明 |
| --- | --- | --- |
| 容器准备 | Partner 前端 | 提供有明确宽高的可见 DOM 容器。 |
| iframe 创建 | Trading Widget SDK | SDK 自动创建并管理 iframe。 |
| 交易 UI | 6MM iframe | K 线、订单簿、下单、持仓、订单和账户数据都在 iframe 内运行。 |
| 显示控制 | Trading Widget SDK + 6MM iframe | PC embed 顶部栏和底部行情跑马灯默认隐藏。Partner 可通过 `display` 开启 shell 区域，或隐藏交易页大容器。 |
| Partner 登录校验 | Partner 后端 | 签发鉴权凭据前，先校验 Partner 自己的用户登录态。 |
| embedToken 签发 | Partner 后端 + Agent API | 推荐用于 partner-token 模式。 |

<h2 id="versioning">版本说明</h2>

公开接入路径保持在 v1 脚本地址。当前 v1 能力包含 partner-token、tokenProvider、entryUrlProvider、display 显示控制、生命周期事件、实例方法和订单事件。

```html
<script src="https://app.6mm.com/sdk/trading-widget/v1/index.js"></script>
```
