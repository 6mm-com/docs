| 当前版本 | 由 6MM 在合作方接入阶段提供确认版本 |

<h2 id="typical-backend-responsibilities">典型后端职责</h2>

- 校验 Partner 用户登录态。
- 将 Partner 用户映射为 agentUserId。
- 用户首次交易或划转前调用 bind。
- 调用 createEntryUrl 生成一次性完整站点 SSO 入口。
- 调用 createEmbedToken 支持 Trading Widget partner-token 模式。
- 校验 Webhook 并做幂等处理。
